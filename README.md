# Board Cutting Game: Greedy vs Divide & Conquer Simulation

## Overview
This project simulates a Board Cutting Game where the goal is to divide a rectangular board into 1x1 unit squares using the minimum number of cuts. It features two algorithms: **Greedy** and **Divide & Conquer**, and allows the user to simulate the process using a graphical interface.

The game provides two modes:

-**Manual Mode:** Where the user selects how the board is split.
-**Auto Mode:** The algorithm automatically splits the board step-by-step.

It helps compare both algorithms on efficiency, complexity, and the optimal number of cuts.

---

## Features
- **Greedy Algorithm** Allows automatic division of the board using a greedy approach.
- **Divide & Conquer Algorithm** Simulates recursive cutting using the divide and conquer strategy.
- **Real-Time User Interaction** Allows users to manually split the board in vertical or horizontal modes.
- **Game Result** Displays the results based on the number of cuts, optimal cuts, and total time taken.
- **Auto Start** Automatically runs the selected algorithm and visualizes the process.
- **Timer**  Measures the time taken to solve the puzzle.

---

---

## Conclusion

This Board Cutting Game is an interactive simulation of the Greedy and Divide & Conquer algorithms. It serves as a tool to compare the efficiency of both algorithms and visualize how they approach solving the problem of cutting a rectangular board into 1x1 unit squares. The game provides valuable insights into the performance of these algorithms and helps users understand algorithmic design and problem-solving techniques
